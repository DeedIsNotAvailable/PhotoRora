[Омский государственный технический университет (OmSTU)](https://omgtu.ru)

Приложение «ИИ-редактор фотографий» для локальной обработки изображений с использованием интеллектуальных инструментов, для устройств на ОС Аврора.

Открывайте снимки из галереи и применяйте ИИ-инструменты, работающие полностью офлайн: удаление и замена фона, улучшение кадра и художественная стилизация. Проект ориентирован на приватную обработку изображений без передачи данных на внешние серверы.

## Сборка проекта

- Установить Aurora IDE / Aurora SDK и запустить Aurora OS Build Engine с нужной целевой платформой.
- До первой CMake-конфигурации проекта установить Conan внутри Build Engine.
- Открыть корень репозитория в Aurora IDE и создать CMake-конфигурацию для выбранного Aurora target.
- Для проверки в эмуляторе использовать конфигурацию Debug, а для получения финального RPM-пакета использовать Release.

## Подпись и установка пакета на устройство

- Настроить подпись пакетов в Aurora IDE или в инструментах Aurora SDK и выбрать нужный профиль сертификата для эмулятора или физического устройства.
- Для самого простого сценария использовать Deploy из Aurora IDE: IDE собирает RPM-пакет, подписывает его выбранным профилем и устанавливает на выбранный эмулятор или устройство.
- Для ручной установки сначала подписать Release RPM-пакет с помощью настроенных инструментов подписи Aurora.
- При необходимости скопировать подписанный пакет на целевое устройство через `scp`, используя SSH-порт и ключ, указанные в настройках эмулятора или устройства в Aurora SDK.
- Установить подписанный пакет на эмуляторе или устройстве стандартным способом через shell целевой системы либо через инструменты deploy в Aurora IDE.
- До окончательной проверки установки сохранить и исходный Release RPM-пакет, и его подписанную версию.
