[Омский государственный технический университет (OmSTU)](https://omgtu.ru)

Приложение «ИИ-редактор фотографий» для локальной обработки изображений с использованием интеллектуальных инструментов, для устройств на ОС Аврора.

Открывайте снимки из галереи и применяйте ИИ-инструменты, работающие полностью офлайн: удаление и замена фона, улучшение кадра и художественная стилизация. Проект ориентирован на приватную обработку изображений без передачи данных на внешние серверы.
